
                           SPIS-LEGACY to SPIS-5.0.0-beta2

                         Global Parameters Converter

                                  V 1.3 - 09/2012

1) Introduction / Purpose

 This script converts the global parameters of a SPIS-Legacy project 
into an XML file compliant with SPIS 5.0.0-beta2 (e.g SPIS-GEO/MEO, SPIS-SCI). 

2) Usage

    globalParametersConverter.sh  pathToLegacySpisProject.spis outputGlobalParam.xml
